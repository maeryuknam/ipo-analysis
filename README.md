# IPO 투자 분석 도구

기술주 전문 애널리스트 관점으로 IPO 기업을 분석하는 웹 도구입니다.
회사명을 입력하면 Claude AI가 **4가지 핵심 관점**으로 투자 분석 리포트를 생성합니다.

## 분석 관점

| # | 관점 | 핵심 질문 |
|---|------|-----------|
| 1 | **내러티브의 현실화** | 진짜 돈이 되는 구조인가? |
| 2 | **해자(Moat)의 본질** | 진짜 기술인가, 마케팅의 결과인가? |
| 3 | **오버행과 밸류에이션** | 누구를 위한 상장인가? |
| 4 | **거시 환경(Macro) 싱크로율** | 시장 타이밍과 외부 리스크 |

각 관점별 긍정·부정·경고·중립 분석과 함께 종합 투자 의견(투자 긍정 / 추가 검토 필요 / 투자 보류)을 제공합니다.

## 화면 구성

- 상단: IPO 분석 프레임워크 (6단계 핵심 분석 영역, Pre-IPO vs 공모청약 비교, Go/No-Go 기준)
- 하단: 회사명 입력 → AI 분석 리포트 실시간 생성

## 설치 및 실행

### 1. 의존성 설치

```bash
npm install
```

### 2. API 키 설정

[Anthropic Console](https://console.anthropic.com)에서 API 키를 발급받은 뒤,
`.env.example`을 복사해 `.env` 파일을 만들고 키를 입력합니다.

```bash
cp .env.example .env
```

```
ANTHROPIC_API_KEY=sk-ant-실제키입력
```

### 3. 서버 실행

```bash
npm start
```

### 4. 접속

브라우저에서 접속:

```
http://localhost:3737/ipo-analysis.html
```

## 기술 스택

- **백엔드**: Node.js + Express
- **AI**: Anthropic Claude API (claude-sonnet-4-6, tool use 구조화 출력)
- **프론트엔드**: 단일 HTML (Vanilla JS, Tabler Icons)

## 비용 안내

분석 1회당 약 $0.01 ~ $0.02의 API 크레딧이 차감됩니다.
HTML 페이지 열람·입력 등은 비용이 발생하지 않으며, "분석 시작" 실행 시에만 과금됩니다.

## 주의

본 도구는 투자 판단을 위한 분석 관점 제공이며, 투자 권유가 아닙니다.
최종 투자 판단의 책임은 사용자 본인에게 있습니다.
